Lists

A list stores multiple items in a single variable. Items are ordered, changeable, and indexed starting from 0.

tools[0]Positive Indexing
tools[-1]Negative Indexing
tools[0:3]Slicing
[::2]Step Slicing
.append()Add to End
.pop()Remove Last
.insert()Insert at Index
.remove()Remove by Value
python 
# Creating a list
tools = ["AWS", "Azure", "GCP", "Docker", "Kubernetes"]

# Indexing
print(tools[0])      # AWS  (first item)
print(tools[-1])     # Kubernetes  (last item)

# Slicing  [start : end : step]
print(tools[0:3])    # ['AWS', 'Azure', 'GCP']
print(tools[::2])    # every 2nd item

# Modifying
tools.append("Git")          # add at end
tools.insert(2, "Terraform") # insert at index 2
tools.pop()                  # remove last item
tools.pop(0)                # remove item at index 0
tools.remove("Git")         # remove by value
💡 IndexError — agar aap kisi aisi index access karo jo exist nahi karta, Python IndexError: list index out of range deta hai.

Dictionaries

Dictionary data key-value pairs mein save karta hai. Keys unique hoti hain, values kuch bhi ho sakti hain.

dict["key"]Access Value
.get()Safe Access
dict["key"]=vAdd/Update
del dict["key"]Delete Key
.update()Bulk Update
.clear()Clear All
python 
# Creating a dictionary
student = {
    "name": "Danish",
    "age":  20,
    "city": "Karachi"
}

# Accessing values
print(student["name"])                   # Danish
print(student.get("grade", "Not Found")) # safe — no error

# Modify / Add / Delete
student["age"]    = 21                   # update existing
student["course"] = "Python"             # add new key
del student["course"]                     # delete key

# Bulk update
student.update({"age": 22, "city": "Lahore"})

# Clear everything
student.clear()

Nested Structures

Lists aur Dictionaries ke andar doosre lists/dictionaries rakh sakte ho — isse Nested Structures kehte hain.

python — nested list 
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matrix[0])       # [1, 2, 3]
print(matrix[1][2])   # 6
print(matrix[2][0])   # 7
python — nested dictionary 
clients = {
    "daniyal": {"name": "Daniyal", "city": "Karachi"},
    "ali":     {"name": "Ali",     "city": "Lahore"}
}
print(clients["daniyal"]["name"])  # Daniyal
print(clients["ali"]["city"])      # Lahore

String Methods

String methods built-in functions hain jo text ke saath kaam karna easy banati hain.

Method Kya Karta Hai Example
.upper() Saari text UPPERCASE kar do "danish".upper() → "DANISH"
.lower() Saari text lowercase kar do "DANISH".lower() → "danish"
.title() Har word ka pehla letter capital "danish abbas".title() → "Danish Abbas"
.capitalize() Sirf pehla letter capital "danish".capitalize() → "Danish"
.strip() Dono sides se spaces hatao " hi ".strip() → "hi"
.replace() Koi substring replace karo "Hi Ali".replace("Ali","Danish")
.split() String ko list mein tod do "a b c".split() → ['a','b','c']
python 
name = "  Danish  Abbas  "

print(name.upper())          # "  DANISH  ABBAS  "
print(name.lower())          # "  danish  abbas  "
print(name.strip())          # "Danish  Abbas"
print(name.title())          # "  Danish  Abbas  "
print(name.replace("Danish", "Ali"))  # "  Ali  Abbas  "
print(name.split())          # ['Danish', 'Abbas']
